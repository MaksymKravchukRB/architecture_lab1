# architecture_lab1

## Getting Started

### Prerequisites

- [.NET SDK 10.0 or later](https://dotnet.microsoft.com/download)
- [Git](https://git-scm.com/)

### Clone the repository

```bash
git clone https://github.com/MaksymKravchukRB/architecture_lab1.git
cd architecture_lab1
```

### Build and run

```bash
dotnet restore
dotnet run
```

The API will start at http://localhost:5077 (or http://localhost:{whatever port got shown to you when you ran the app}). Swagger UI is available at /swagger.
